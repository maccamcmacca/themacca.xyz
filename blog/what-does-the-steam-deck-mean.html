What does the Steam Deck mean?

Valve recently announced their latest hardware product, the Steam Deck. A handheld “gaming PC” similar in shape and function to a Nintendo Switch except it isn’t a locked down OS. In fact it will be running the new SteamOS which apparently is now based on Arch Linux as opposed to the previous Debian based attempts.

A lot of Linux users are showing interest in the device, which is understandable given it is a unique product in the Linux space.

There seems to be a consensus amongst some online personalities that the Steam Deck is going to be good for Linux or that it will finally convince the “normies” to give Linux a try.

While this is something that I would like to see, I’m not convinced and I’m not sure the people hyping such an idea are either.

Hardware:

These devices have some impressive hardware under the hood, especially considering that it will be competing in the same space as the Nintendo Switch. All three devices have the same computing power (on paper), they differ in storage and display quality.

The lowest end configuration is effectively restricted to small indie games, emulation and slightly older titles. This is due to the storage provided just not being large enough for the large install size of modern games. Now this isn’t necessarily a bad thing, in fact it’s almost the perfect competitor to the Switch at it’s price point and with the open nature of its Linux based OS it will likely be the emulation king for a long time.

Higher end models will be interesting to keep an eye on due to their more suitable storage capacities and speeds as well as the highest end model having a better display.

Including joysticks alongside touch panels is a great decision as it makes the device suitable for both traditional PC genres and normal computing tasks in addition to gamepad compatible games.

The integrated display resolution of 1280x800 is a bit low but it makes sense for a portable device inteded for games. There is a reason the Nintendo Switch doesn’t have a 1080p panel and it’s the same reason the Steam Deck doesn’t. Additionally the device apparently targets 30fps in higher performance games, this is about as much as should be reasonably expected from the device. It’s a non-issue as far as I am concerned as games that are more suitable for this platform should run perfectly fine at 60fps.

Steam Machines:

This product is effectively the first true “Steam Machine” Linux gaming PC. Which was a project from almost a decade ago that was cancelled but has now apparently made a return as a handheld device.

Valve is bringing back SteamOS, which will now be an Arch Linux based distro which is driving Valve’s dedication to getting anti-cheat software to work with Proton. However this is a double edged sword as it effectively eliminates the need for developers to compile and optimise for Linux.

Another positive is that it is a relatively inexpensive way (current hardware prices) for gamers to get familiar with the idea of using Linux whilst giving them an out of the box working device with (supposedly) no need to tinker.

I personally see no real negatives for the consumer from the development of this device.

Something to note is that Linux users are not Valve’s intended audience for this product. This is a good thing.

The choice of a handheld instead of a desktop PC was a very clever idea. This allows gamers to get their feet wet with Linux without replacing their existing hardware.

The Windows Virus:

Do I think this is going to be the “Windows killer” that it is being hyped up as? No. The “Windows only exists due to gaming” meme is just a meme.

However if this does bring more people over to Linux on their main computers (and that is the important part) then the device will be a success as far as I’m concerned. Do I see this happening at any kind of scale? No.

The fact the device will be Linux out of the box is a bold move and I respect that, although now anyone with a glorified GameBoy can now claim “I run Arch BTW” and that meme is officially dead.

Alternative Uses:

The topic of emulation has already came up a lot regarding this device and I see it being popular for this reason alone.

The device will support the installation of a Windows operating system if the owner chooses so. While it would be great if they didn’t it is inevitable that some people just won’t want to use Linux. On the bright side though there will be users that do.

One thing that I haven’t seen much discussion about that seems more important and relevant is that in addition to being a games console this device is also a powerful Linux tablet that can compete in the tablet space.

If the device can handle a decent workload it may even function as a desktop replacement for some users that don’t care for high end graphical fidelity. After all it is running a full Linux OS and supports package installation.

The Hype Issue:

It seems pretty clear that a lot of the hype around the Steam Deck is coming from people that are setting themselves up for disappointment. Which is inevitable anyway due to the nature of hyping a product.

This is not a Gaming PC replacement, this is a gaming PC supplement. You will only really benefit from owning this device if you own a gaming PC already.

Don’t pre-order. You have literally no reason to do this. Ever.
